## 아이디어 1: Cyberpunk Data Terminal (사이버펑크 데이터 터미널)

**Design Movement**: Cyberpunk / Sci-Fi HUD Interface
**Core Principles**:
1. 네온 글로우 효과가 적용된 다크 인터페이스로 미래지향적 분위기
2. 데이터 시각화 중심 — 수치와 차트가 주인공
3. 스캔라인, 그리드 패턴, 홀로그램 효과로 기술적 깊이감

**Color Philosophy**: 
- 배경: #0a0e1a (깊은 우주 네이비)
- 시안 네온: #00f5ff (주요 강조)
- 퍼플 글로우: #a855f7 (보조 강조)
- 텍스트: #e2e8f0 (밝은 실버)

**Layout Paradigm**: 
- 풀스크린 세로 스크롤, 각 섹션이 터미널 화면처럼 전환
- 좌측 고정 네비게이션 바 (HUD 스타일)

**Signature Elements**: 
1. 네온 보더 글로우 카드
2. 스캔라인 오버레이 텍스처

**Interaction Philosophy**: 스크롤 시 섹션이 "부팅"되는 느낌의 페이드인

**Animation**: 타이핑 효과, 글로우 펄스, 스크롤 트리거 페이드인

**Typography**: Space Grotesk (제목) + Inter (본문)

## 아이디어 2: Financial Dashboard Brutalism (금융 대시보드 브루탈리즘)

**Design Movement**: Neo-Brutalism + Financial Dashboard
**Core Principles**:
1. 두꺼운 보더와 강렬한 색상 블록으로 정보 구분
2. 데이터 테이블과 수치가 RAW하게 노출
3. 의도적으로 거친 타이포그래피로 신뢰감 형성

**Color Philosophy**:
- 배경: #1a1a2e (다크 인디고)
- 라임 그린: #39ff14 (수익/긍정 지표)
- 핫 핑크: #ff006e (위험/주의 지표)
- 옐로우: #ffd60a (하이라이트)

**Layout Paradigm**:
- 비대칭 그리드, 정보 밀도가 높은 대시보드 스타일
- 섹션마다 다른 그리드 구조

**Signature Elements**:
1. 두꺼운 4px 솔리드 보더
2. 오프셋 그림자 (brutalist shadow)

**Interaction Philosophy**: 클릭/호버 시 색상 반전 효과

**Animation**: 숫자 카운트업, 바 차트 성장 애니메이션

**Typography**: DM Sans Black (제목) + Work Sans (본문)

## 아이디어 3: Holographic Glass Morphism (홀로그래픽 글래스모피즘)

**Design Movement**: Glassmorphism + Holographic Gradient
**Core Principles**:
1. 반투명 글래스 패널 위에 콘텐츠가 떠있는 레이어드 구조
2. 홀로그래픽 그라데이션이 배경에서 은은하게 흐르는 프리미엄 느낌
3. 블러와 투명도로 깊이감 있는 Z축 레이아웃

**Color Philosophy**:
- 배경: #0f0f23 (딥 미드나잇)
- 홀로그래픽: linear-gradient(135deg, #667eea, #764ba2, #f093fb)
- 글래스: rgba(255,255,255,0.05) + backdrop-blur
- 텍스트: #ffffff (화이트)

**Layout Paradigm**:
- 중앙 정렬 카드 스택, 각 섹션이 글래스 패널
- 배경에 큰 홀로그래픽 오브가 천천히 움직임

**Signature Elements**:
1. Frosted glass 카드 (backdrop-blur-xl)
2. 홀로그래픽 그라데이션 오브

**Interaction Philosophy**: 마우스 움직임에 따라 글래스 반사각 변화

**Animation**: 부드러운 패럴랙스, 글래스 패널 슬라이드인

**Typography**: Poppins (제목) + Inter (본문)
